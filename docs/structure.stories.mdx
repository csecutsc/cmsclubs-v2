# Folder Structure

### Learn about the codebase

<div style={{ margin: `2rem 0` }} />

## Root of repo

| Folder/File        | Description                                                                               |
| ------------------ | ----------------------------------------------------------------------------------------- |
| `codegen`          | Contains pre-build code for generating code for stuff                                     |
| `docs`             | Contains pure documentation for storybook (like this page)                                |
| `src`              | Your code                                                                                 |
| `static`           | Static files that are served. As of now, it's optional and generated by codegen for fonts |
| `.env.example`     | Example of what your `.env` should look like                                              |
| `gatsby-config.js` | If you need explanation, I recommend a Gatsby tutorial before this                        |

<div style={{ margin: `3rem 0` }} />

## `src` folder

| Folder       | Description                                                                                                                  |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| `components` | All reusable components, which are on this component explorer and more                                                       |
| `images`     | "Static" images which would be processed by `gatsby-image-sharp`                                                             |
| `pages`      | Gatsby specific folder. Learn Gatsby on your own free time.                                                                  |
| `plugins`    | WIP, might not exist in the repo depending on when you're reading this. Contains components related to site creation feature |
| `sections`   | Sections of the pages in the `/pages` folder. It should follow the structure of `/sections/[PAGE NAME]/...`.                 |
| `utils`      | Reusable shared stuff like styles, hooks, propTypes...                                                                       |
