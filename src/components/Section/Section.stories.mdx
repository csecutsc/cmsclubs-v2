# Section Component
### Basic building block for sections

<div style={{ margin: `4rem 0` }}/>

## Description

This component is **always** the root component for any component within `/pages`
as it defines the maxWidth of page and appropriate spacing to ensure a consistent look.

The anonamy of the component is as follows,
```
 ___________________
|     Container     |
|   _____________   |
|  |   Content   |  |
|  |_____________|  |
|___________________|
```
